## 📍 Commit Rules
* 백준/프로그래머스 허브를 이용해 커밋한다. 
* 당일 스터디에 올렸던 코드들을 정오까지 레포지토리에 커밋하고 pull request한다.
* 다른 사람들의 PR을 확인한 뒤, 자유롭게 코드리뷰를 작성한다. 
* 스터디가 끝난 뒤, 수정된 코드를 Merge한다.
